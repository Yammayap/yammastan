# yammastan

A PHPStan extension to ignore common issues.

## Installation

```bash
composer require yammayap/yammastan --dev
```

## Usage
Add the following to your `phpstan.neon` file:

```neon
includes:
    - ./vendor/yammayap/yammastan/extension.neon
```
